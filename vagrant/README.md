# Vagrant cheat
```
vagrant box add ubuntu/focal64 

cd vms

vagrant init ubuntu/focal64 

vim Vagrantfile 

vagrant up 

ssh 127.0.0.1:2222 

ssh -p 2222 127.0.0.1 

ssh -p 2222 vagrant@127.0.0.1 

vagrant halt 

vagrant plugin install vagrant-vbguest 

vagrant reload 

vagrant ssh 

vagrant halt 
```
